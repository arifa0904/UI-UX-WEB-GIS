# Tools untuk UI/UX Designer & Front-End WebGIS
### 🎨 UI/UX Design
1. Figma
Kegunaan:
- Membuat wireframe dan prototype
- Mendesain user interface (UI)
- Membuat design system
- Kolaborasi tim secara real-time
Digunakan untuk merancang tampilan WebGIS sebelum diimplementasikan ke dalam kode.
2.FigJam
Kegunaan:
- Brainstorming
- User journey mapping
- Flowchart sistem ( jika dibutuhkan )
Digunakan untuk merancang struktur sistem WebGIS sebelum tahap desain visual.

----

### 💻 Front-End
1. HTML
Kegunaan:
- Struktur dasar halaman web
- Membuat elemen peta, sidebar, navigasi, dan dashboard
2. CSS
Kegunaan:
- Styling tampilan WebGIS
- Layouting (Flexbox, Grid)
- Responsive design
3.JavaScript
Kegunaan:
- Membuat interaksi dinamis
- Manipulasi data peta
- Event handling (zoom, klik marker, popup informasi)
4. React.js / Vue.js
Kegunaan:
- Membangun tampilan berbasis komponen
- Mengelola state aplikasi
- Membuat dashboard WebGIS lebih terstruktur dan scalable
